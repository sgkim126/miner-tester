# miner-tester
A miner tester

## usage
```
usage: tester.py [-h] [--difficulty DIFFICULTY] [--port PORT] [--miner MINER]
                 [--repeat REPEAT] [--seed SEED]

optional arguments:
  -h, --help            show this help message and exit
  --difficulty DIFFICULTY
                        A difficulty in hex string. (e.x. 0x7fffffffffffffffff
                        ffffffffffffffffffffffffffffffffffffffffff)
  --port PORT           A listening port (default: 8080)
  --miner MINER         A miner port (default: 3333)
  --repeat REPEAT       The number of repeat (default: 10)
  --seed SEED           A random seed (default: 0)
```
